Image Processing & Binary Data Conversion

Project Overview

This project focuses on digital image processing techniques, specifically working with binary images. Binary images are fundamental in computer vision and image processing, containing only two colors: black (0) and white (1). The project implements various image processing algorithms to convert and manipulate images, with a focus on practical applications in computer vision and machine learning.

Technical Implementation

1. Image Preprocessing

Before converting an image to binary format, we perform several preprocessing steps to ensure optimal results:

import cv2
import numpy as np

def preprocess_image(image):
    # Convert to grayscale
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    
    # Apply Gaussian blur to reduce noise
    blurred = cv2.GaussianBlur(gray, (5, 5), 0)
    
    # Normalize the image
    normalized = cv2.normalize(blurred, None, 0, 255, cv2.NORM_MINMAX)
    
    return normalized

2. Image Thresholding

Thresholding is the process of converting a grayscale image into a binary image. This is done by setting a threshold value, where pixels above the threshold become white (1) and pixels below become black (0). We implement both global and adaptive thresholding methods:

def threshold_image(image, method='global', threshold=127):
    if method == 'global':
        _, binary = cv2.threshold(image, threshold, 255, cv2.THRESH_BINARY)
    elif method == 'adaptive':
        binary = cv2.adaptiveThreshold(
            image, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
            cv2.THRESH_BINARY, 11, 2
        )
    return binary

3. Morphological Operations

Morphological operations are used to process binary images based on shapes. These operations help in noise removal, object detection, and feature extraction:

Dilation

Dilation adds pixels to the boundaries of objects in an image. It's useful for filling gaps between objects and connecting broken parts:

def dilate_image(binary_image, kernel_size=3, iterations=1):
    kernel = np.ones((kernel_size, kernel_size), np.uint8)
    dilated = cv2.dilate(binary_image, kernel, iterations=iterations)
    return dilated

Erosion

Erosion removes pixels from the boundaries of objects. It's useful for removing noise and small objects, and for separating connected objects:

def erode_image(binary_image, kernel_size=3, iterations=1):
    kernel = np.ones((kernel_size, kernel_size), np.uint8)
    eroded = cv2.erode(binary_image, kernel, iterations=iterations)
    return eroded

Opening and Closing

Opening and closing are combinations of erosion and dilation operations that help in noise removal and object detection:

def morphological_operations(binary_image, kernel_size=3):
    kernel = np.ones((kernel_size, kernel_size), np.uint8)
    
    # Opening: erosion followed by dilation
    # Useful for removing small objects and noise
    opening = cv2.morphologyEx(binary_image, cv2.MORPH_OPEN, kernel)
    
    # Closing: dilation followed by erosion
    # Useful for filling small holes and connecting nearby objects
    closing = cv2.morphologyEx(binary_image, cv2.MORPH_CLOSE, kernel)
    
    return opening, closing

4. Feature Extraction

After processing the binary image, we can extract various features for analysis:

def extract_features(binary_image):
    # Find contours
    contours, _ = cv2.findContours(
        binary_image, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE
    )
    
    features = []
    for contour in contours:
        # Calculate area
        area = cv2.contourArea(contour)
        
        # Calculate perimeter
        perimeter = cv2.arcLength(contour, True)
        
        # Calculate moments
        moments = cv2.moments(contour)
        
        features.append({
            'area': area,
            'perimeter': perimeter,
            'moments': moments
        })
    
    return features
